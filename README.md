# Проектирование UI/UX

# 1. Описание проекта

**FlashLearn** — веб-приложение, позволяющее пользователю загружать PDF-конспекты и автоматически генерировать учебные карточки формата **вопрос-ответ** для эффективного повторения материала.

Основные функции:

- регистрация и авторизация
    
- загрузка PDF
    
- генерация карточек
    
- просмотр карточек
    
- режим обучения
    

---

### Сценарий 2 — загрузка PDF

1. Пользователь заходит в Dashboard
    
2. Нажимает **Upload PDF**
    
3. Выбирает файл
    
4. Система обрабатывает документ
    
5. Появляется набор карточек
    

---

### Сценарий 3 — изучение карточек

1. Пользователь открывает набор карточек
    
2. Видит вопрос
    
3. Нажимает **Show Answer**
    
4. Переходит к следующей карточке
